# Question 1 - Shell Script for Duplicate Submission Management

## Objective
Detect duplicate student submissions using MD5 hashing, back up unique files, and generate a report.

## Files
- submission_manager.sh
- report.txt
- errors.log

## Execution

```bash
chmod +x submission_manager.sh
./submission_manager.sh
```

## Sample Output

Files Processed: 3
Duplicate Files: 1
Backed Up Files: 2
